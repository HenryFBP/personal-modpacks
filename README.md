# Personal Modpacks

## Caveats

- You cannot have Fabric auto-installed with MultiMC. See <https://fabricmc.net/wiki/install#multimc>.

## Building

    cd modpack-folder

    packwiz cf export

    ls export.zip

## Using a modpack

1.  Import `export.zip` into your MultiMC/CurseForge/whatever.
2.  See [Caveats](#Caveats).
